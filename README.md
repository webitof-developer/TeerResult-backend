# Backend for TeerGame

This is the backend API server for the TeerGame application, built with Node.js and Express.js.

## Features

- User authentication and management
- Game management
- Result management
- Banner management
- Contact form handling
- Admin settings

## Installation

1. Clone the repository
2. Navigate to the backend directory
3. Install dependencies: `npm install`
4. Set up environment variables in `.env` file
5. Run the server: `npm start`

## API Endpoints

- `/api/users` - User management
- `/api/games` - Game operations
- `/api/results` - Result handling
- `/api/banners` - Banner management
- `/api/contacts` - Contact forms
- `/api/admin-settings` - Admin configurations

## Technologies Used

- Node.js
- Express.js
- MongoDB (or your database)
- JWT for authentication
- Multer for file uploads
